## 🚀 Як користуватись документацією

### Для українськомовних користувачів 🇺🇦
1. Відкрийте основний файл (напр., `README.md`)
2. Клацніть на "🇺🇦 Українська версія"
3. Читайте документацію на рідній мові

### Для англомовних користувачів 🇬🇧
1. Відкрийте основний файл (напр., `README.md`)
2. Клацніть на "🇬🇧 English Version"
3. Читайте документацію англійською мовою

### Для гіт-репозиторію
Усім користувачам рекомендується починати з основних файлів:
- `README.md` - для инструкцій запуску
- `docs/WORK_PLAN.md` - для розуміння плану розвитку
- `docs/SPRINT_STATUS.md` - для статусу проєкту
